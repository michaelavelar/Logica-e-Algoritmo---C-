Combina duas ou mais queries, retornando os elementos juntos

SELECT * colunas FROM TABELA
UNION
SELECT * colunas FROM TABELA_2

NESTE CASO COMANDO RETIRA AS DUPLICIDADES

===

SELECT * colunas FROM TABELA
UNION ALL
SELECT * colunas FROM TABELA_2

NESTE CASO COMANDO RETORNA TODOS OS VALORES INCLUSIVE COM DUPLICIDADE
